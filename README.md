# Bruna Soares · MS Ads Intelligence

Relatório dinâmico público de Meta Ads para Manipulação Silenciosa.

- Período: 2025-12-01 a 2026-02-28
- Contas: CONTA - DM e CONTA - DM[2]
- Base canônica: `purchase`, `initiate_checkout`, `landing_page_view`, sem somar `omni_purchase`/pixel duplicado.
- V3: tabela padrão agrupada somente por `ad_name` idêntico. Numerações diferentes como MSAD208, MSAD260 e MSAD261 não são consolidadas entre si.
- Não contém tokens ou credenciais.
